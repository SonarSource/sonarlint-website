SonarLint for Visual Studio version 2.8

  • Release date: November 5, 2016
  • Release notes for SonarLint
  • Release notes for C#
  • Download Extension

This version of SonarLint brings the Connected Mode for VB.NET projects. A major improvement for C# is that we enhanced the symbolic execution engine to resolve transitive dependencies between local variables. Also, some rules have been tweeked, such as S2355, which does not report false positives when array size is explicitly defined. Furthermore, S1144 has been improved to not report when method references can be resolved to multiple candidate declarations. Finally, here's are the new rules since version 2.7:

  • Short-circuit logic should be used in boolean contexts - Rule S2178 (C# and VB.NET)
  • Report on issues being suppressed in the source - Rule S1309 (C#)
