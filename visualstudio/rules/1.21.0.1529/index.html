SonarLint for Visual Studio version 2.9-RC1

  • Release date: January 9, 2017
  • Release notes for SonarLint
  • Release notes for rules

This version of SonarLint enables support of VS2017 and brings many C# rules to the IDE. We also fixed some bugs with the dataflow engine and have improved multiple rules. Here are the new rules since version 2.8.1:

  • Exceptions should not be thrown in finally blocks - Rule S1163
  • General exceptions should never be thrown - Rule S112 [C#, VB.Net]
  • Static fields should not be updated in constructors - Rule S3010
  • IComparable should be implemented together with other comparison methods - Rule S1210
  • "String.IsNullOrEmpty" should be used - Rule S3256
  • "NaN" should not be used in comparisons - Rule S2688
  • Track lack of copyright and license headers - Rule S1451
