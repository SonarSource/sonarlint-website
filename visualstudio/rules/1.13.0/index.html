  • Release notes for C#
  • Download Extension

This version adds some small improvements to the connected mode, such as SonarQube porject list scrolling in the binding view. Also, SonarLint version 2.2.1 adds one new rule to the product, and 17 many rules were improved or fixed to not report false positives and handle more cases. Here are the major changes since version 2.2:

  • Nullable type comparison should not be redundant - Rule S3610 (new)
  • Redundant casts should not be used (Rule S1905) handles casts with as
  • Runtime type checking should be simplified (Rule S2219) adds support for as and is
  • Method overrides should not change parameter defaults (Rule S1006) correctly handles explicit interface implementations
  • Inheritance list should not be redundant (Rule S1939) doesn't report false positives
