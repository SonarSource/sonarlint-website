SonarLint for Visual Studio version 2.5

  • Release date: July 27, 2016
  • Release notes for SonarLint
  • Release notes for C#
  • Download Extension

One new rule and several improvements were added to SonarLint for Visual Studio version 2.5. The improvements include the extension of Rule S1944 to report on nullable casts that definitely fail due to the nullness of the operand. User reported false positives were handled, for example Rule S1858 doesn't report on structs anymore, Rule S3459 doesn't report on fixed arrays anymore. Also, the symbolic execution engine was improved to propagate knowledge through complex expressions. Finally, here is the new rule since version 2.4:

  • Empty nullable value should not be accessed - Rule S3655
