In this version, we have added three new rules, improved Rule S1854, and fixed Rule S2333 to take into account +=-like assignments when checking for redundant checked and unchecked statements. Here are the new rules since version 2.2.1:

  • Recursion should not be infinite - Rule S2190
  • Member initializer values should not be redundant - Rule S3604
  • Jump statements should not be redundant - Rule S3626
